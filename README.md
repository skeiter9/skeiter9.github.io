# My CV ;)

# Install

```
npm install
```

## Generate Resume commands

```bash
npm run build-pdf # => npx resume export resume.pdf --theme caffeine",
npm run build-web # => npx resume export index.html --theme caffeine",
npm run build # => npx npm run build-pdf && npm run build-web"
```

### Notes

- version 2022 use ["Caffeine"](https://jsonresume.org/themes/) theme
